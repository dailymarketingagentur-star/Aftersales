## 2. Health Score (monatlich bewerten)

### Bewertungskategorien

Jede Kategorie wird auf einer Skala von 1-5 bewertet:
- **1** = Kritisch (sofort handeln)
- **2** = Schlecht (Verbesserungsplan noetig)
- **3** = Neutral (beobachten)
- **4** = Gut (halten)
- **5** = Exzellent (Testimonial/Referral-Chance)

| Kategorie | Score (1-5) | Notizen |
|---|---|---|
| **Ergebnis-Zufriedenheit:** Erreichen wir die vereinbarten KPIs? | | |
| **Kommunikation:** Antwortet der Kunde zeitnah? Nimmt er an Calls teil? | | |
| **Engagement:** Nutzt der Kunde Dashboard, Portal, gibt Feedback? | | |
| **Beziehung:** Wie ist die zwischenmenschliche Dynamik? Vertrauen? | | |
| **Zahlungsverhalten:** Puenktliche Zahlung? Keine Rueckfragen zu Rechnungen? | | |
| **Wachstumspotenzial:** Upsell-/Cross-Sell-Moeglichkeiten erkennbar? | | |
| **Referral-Bereitschaft:** Wuerde der Kunde uns empfehlen? | | |
| **Gesamt-Score** | **/35** | |

### Score-Interpretation

| Gesamtscore | Status | Aktion |
|---|---|---|
| 30-35 | Exzellent | Testimonial anfragen, Referral-Gespraech fuehren |
| 24-29 | Gut | Wow-Moment planen, Upsell-Chance pruefen |
| 18-23 | Neutral | Proaktiv Feedback einholen, Verbesserungen identifizieren |
| 12-17 | Risiko | Sofort Massnahmen-Plan erstellen, Geschaeftsfuehrer einbeziehen |
| 7-11 | Kritisch | Rettungs-Meeting innerhalb 48h, Exit-Prevention aktivieren |

## 5. Churn-Warnsignale

> Visuelle Darstellung: [diagramme/04-churn-warning-entscheidungsbaum.md](../diagramme/04-churn-warning-entscheidungsbaum.md)

### Fruehe Warnsignale (sofort handeln!)

- [ ] Kunde antwortet langsamer als gewoehnlich auf Emails/Nachrichten
- [ ] Kunde verpasst Check-in Calls oder verschiebt wiederholt
- [ ] Nutzung von Dashboard/Portal sinkt deutlich
- [ ] Kunde fragt nach Vertragsbedingungen/Kuendigungsfristen
- [ ] Neuer Entscheider/Ansprechpartner beim Kunden
- [ ] Kunde erwaehnt Budgetkuerzungen
- [ ] Wettbewerber werden erwaehnt oder verglichen
- [ ] Feedback wird kritischer oder bleibt ganz aus
- [ ] Zahlungen verzoegern sich
- [ ] Weniger Freigabe-Anfragen (= weniger Engagement)

### Reaktionsplan bei Warnsignalen

| Signal | Sofort-Massnahme | Innerhalb 1 Woche |
|---|---|---|
| Antwortet langsamer | Persoenlicher Anruf (nicht Email) | Vor-Ort-Meeting anbieten |
| Verpasst Calls | Flexibleren Termin anbieten, Agenda vorab senden | Wert der Calls verdeutlichen |
| Dashboard-Nutzung sinkt | Kurze Video-Einweisung senden | Dashboard vereinfachen |
| Fragt nach Kuendigung | Geschaeftsfuehrer einbeziehen | Verbesserungsplan praesentieren |
| Neuer Ansprechpartner | Kennenlern-Call vereinbaren | Erfolge neu praesentieren |
| Budget-Kuerzung | Flexibles Modell anbieten | Downgrade statt Kuendigung |
| Wettbewerber-Vergleich | USPs hervorheben, Ergebnisse zeigen | Exklusives Angebot erstellen |

## 6. Exit-Prevention Protokoll

### Bei Kuendigungsversuch

1. **Sofort:** Nicht defensiv reagieren. "Danke, dass du offen mit uns sprichst."
2. **Innerhalb 24h:** Exit-Interview vereinbaren (Video-Call, 30 Min)
3. **Exit-Interview Fragen:**
   - "Was ist der Hauptgrund fuer die Kuendigung?"
   - "Gibt es etwas, das wir haetten anders machen koennen?"
   - "Wenn wir [konkretes Problem] loesen -- wuerde das etwas aendern?"
   - "Waere ein anderes Leistungspaket/Preismodell eine Option?"
4. **Angebote (gestaffelt):**
   - Option A: Verbesserungsplan mit konkreten Massnahmen
   - Option B: Downgrade auf kleineres Paket
   - Option C: Pause statt Kuendigung (1-3 Monate)
5. **Falls Kuendigung final:** Professionell abschliessen, Tuer offen lassen
6. **Win-Back:** Nach 3 und 6 Monaten erneut Kontakt aufnehmen

### Win-Back Email-Template

**Betreff:** Wir vermissen die Zusammenarbeit, {{KUNDENNAME}}

## 7. Quarterly Health Review (intern)

### Agenda fuer internes Team-Meeting (30 Min, quartalsweise)

1. Portfolio-Ueberblick: Alle Kunden-Scores auf einen Blick
2. Rote Flaggen: Kunden mit Score < 18 besprechen
3. Erfolge: Top-Kunden hervorheben, Learnings teilen
4. Upsell-Pipeline: Welche Kunden sind bereit fuer mehr?
5. Referral-Potenzial: Wer ist bereit fuer Testimonials/Referrals?
6. Prozess-Verbesserungen: Was koennen wir besser machen?
